# Operation in Python

# Aritmetics
 - \+ : penambahan
 - \- : pengurangan
 - \* : pengkalian
 - \/ : pembagian
 - ** : pangkat
 - \// : pembulatan
 - \% : modulus

 ## Assignment

 - += : menambahkan nilai
 - -= : mengungkan nilai

 ## Fungsi
- () : pernyataan atau statement
- {} : diksi atau referensi

## Logika
- and
- or
- not

## Comparison
- == : sama dengan
- \> : lebih besar
- \>= : lebih besar sama dengan
